# ⚖️ 罕见事件与非线性相互作用治理红线

1. **评估指标红线**：
   - 严禁使用 Accuracy 评估模型！必须计算并汇报 **PR-AUC (Precision-Recall 曲线下面积)**、**Recall (召回率)** 与 **F2-Score**（针对罕见病漏诊实施高惩罚）。

2. **不平衡数据采样规范**：
   - 在拟合传统模型前，必须使用 `imbalanced-learn` 的 `BorderlineSMOTE` 或调整 `scale_pos_weight` 应对样本倾斜。

3. **信号挖掘与非线性解释规范**：
   - 必须先计算并输出 **报告比值比 (ROR - Reporting Odds Ratio)** 及其 95% 置信区间。
   - 必须使用 **SHAP** 绘制多药联合使用的非线性交互依赖图 (SHAP Interaction Plot)，证明模型捕捉到了非线性相互作用。
