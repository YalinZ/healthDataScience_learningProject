# ⚖️ 生存分析 (Survival Analysis) 核心行为红线

1. **缺失值处理红线**：
   - 肺癌数据集中的 `meal.cal` 或 `wt.loss` 可能包含缺失值，必须先统计缺失比例。严禁无脑直接 `dropna()` 丢弃导致样本量骤减，需明确告知用户处理策略。

2. **统计图表规范**：
   - 绘制 Kaplan-Meier 生存曲线时，**必须带上 95% 置信区间 (Confidence Intervals)**。
   - 必须通过 Log-Rank 检验计算 p-value，并在图表或 Markdown 中标注显著性（p < 0.05）。

3. **Cox 比例风险模型 (Cox PH) 规范**：
   - 汇报 Cox 模型结果时，不能仅看回归系数，**必须计算并解释风险比 (Hazard Ratio, HR)** 及其 95% 置信区间。
   - 解释 HR 含义：例如 HR > 1 代表该因素增加死亡风险，HR < 1 代表该因素为保护性因素。

4. **机器学习生存分析 (RSF) 规范**：
   - 必须使用 `sksurv.ensemble.RandomSurvivalForest` 训练随机生存森林模型。
   - 评估指标强制统一使用 **C-index (Concordance Index)**，严禁使用普通的 Accuracy 或 ROC-AUC。
   - 必须将数据集划分为 8:2 训练集与测试集，并在测试集上对比传统 Cox 模型与 RSF 模型的 C-index 表现。
