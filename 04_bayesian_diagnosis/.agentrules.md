# ⚖️ 贝叶斯信念网络 (BBN) 建模与推理红线

1. **概率完备性红线**：
   - 每一个节点的条件概率表 (CPT) 元素之和必须严格等于 1.0，严禁出现概率不守恒。

2. **推理引擎规范**：
   - 必须使用 `pgmpy.models.BayesianNetwork` 构建有向无环图 (DAG)。
   - 必须使用 `pgmpy.inference.VariableElimination` (变量消除法) 进行后验概率推导。

3. **动态更新展示要求**：
   - 必须展示在**无症状（仅先验概率）**、**引入症状 1（如发热）**、**引入关键症状 2（如味觉丧失）** 三个阶段下，各类疾病后验概率分布的柱状图变化。

4. **模型对比与实验规范 (Naive Bayes vs BBN)**：
   - 必须使用 `scikit-learn` 的 `BernoulliNB` 训练一个朴素贝叶斯分类器作为 Baseline。
   - 在相同证据（如 `Fever=1, LossOfTaste=1`）下，必须同时打印出朴素贝叶斯与 BBN 算出的疾病概率分布。
   - 必须在 Markdown 单元格中总结：朴素贝叶斯的“特征条件独立”假设在复杂多症状诊断中会导致什么偏见或失真。
